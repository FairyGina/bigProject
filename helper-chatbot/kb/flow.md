# BeanRecipe(빈 레시피) 홈페이지 FAQ / 사용방법 KB (flow.md)
# - 목적: 사용자가 “어디서 뭐 해?” 물어보면, 정확한 ‘메뉴 경로 + 버튼/입력 + 결과’를 안내
# - 작성 규칙:
#   - 섹션(##) 하나 = 화면/기능 하나
#   - “메뉴 경로”는 좌측 사이드바 기준
#   - “키워드”는 검색/매칭용(많을수록 좋음)

## 공지사항 (홈페이지 공지사항)
- 메뉴 경로: 좌측 메뉴 > **공지사항**
- 화면 설명:
  - 공지사항 게시판 형태(테이블)로 **번호 / 제목 / 작성자 | 날짜**가 보임
  - 하단에 **검색(제목 드롭다운 + ‘공지사항 검색’ 입력창)**, **페이지네이션(< 1 >)** 제공
  - 우측 하단에 **‘글 작성’** 버튼이 있음(권한 있는 사용자만 작성 가능)

- 사용 방법:
  1) 좌측 메뉴에서 **공지사항** 클릭  
  2) 목록에서 원하는 공지글을 **제목 클릭**해 확인  
  3) 검색하려면 하단에서 조건(기본: 제목) 선택 → 검색어 입력  
  4) 작성은 우측 하단 **글 작성**(권한 필요)

- 질문 예시/답변 템플릿:
  - Q: 공지사항은 어디서 봐?
    - A: 좌측 메뉴에서 **공지사항**을 누르면 “홈페이지 공지사항” 목록이 나와요. 거기서 제목을 클릭하면 확인할 수 있어요.
  - Q: 공지사항 검색은 어떻게 해?
    - A: 화면 하단의 검색 영역에서 (기본: 제목) 선택 후 검색어를 입력하면 돼요.
  - Q: 공지사항 글 작성은 어디서 해?
    - A: 우측 하단 **글 작성** 버튼에서 작성할 수 있어요(계정 권한이 있어야 보이거나 동작할 수 있어요).

- 키워드: 공지사항, 게시판, 글작성, 글 작성, 검색, 제목, 작성자, 날짜, 페이지, 페이지네이션, 목록

## 레시피 허브 (전체 레시피)
- 메뉴 경로: 좌측 메뉴 > **레시피 허브**
- 화면 설명:
  - 상단 제목: **전체 레시피**
  - “제목 검색” 입력창(placeholder: “제목으로 레시피를 검색합니다”)
  - 레시피가 **카드(썸네일 이미지 + 레시피명)** 로 표시됨
  - 우측 상단에 **라이트 모드** 토글 버튼이 있을 수 있음

- 사용 방법:
  1) 좌측 메뉴 **레시피 허브** 클릭  
  2) 전체 레시피가 카드로 보임  
  3) 찾고 싶은 레시피는 상단 검색창에 **제목 키워드 입력**  
  4) 카드 클릭 → **레시피 상세 화면**으로 이동

- 질문 예시/답변 템플릿:
  - Q: 레시피 허브에서 전체 레시피는 어떻게 찾아?
    - A: 좌측 메뉴에서 **레시피 허브**를 누르면 “전체 레시피” 화면이 나오고, 카드로 레시피를 볼 수 있어요. 상단 검색창으로 제목 검색도 가능해요.

- 키워드: 레시피 허브, 전체 레시피, 제목 검색, 카드, 썸네일, 레시피 목록, 검색

## 유저 허브 (내 정보 수정)
- 메뉴 경로: 좌측 메뉴 > **유저 허브** > **내 정보 수정**
- 화면 설명:
  - 상단 제목: **내 정보 수정**
  - 좌측 “기본 정보”: 아이디(이메일), 이름, 생년월일 입력
  - 우측 “비밀번호 변경”: **SNS 계정은 변경할 수 없습니다** 안내가 표시될 수 있음
  - 하단 버튼: **취소 / 수정 완료**

- 사용 방법:
  1) 좌측 메뉴에서 **유저 허브** 펼치기  
  2) **내 정보 수정** 클릭  
  3) 이름/생년월일 등 수정 → **수정 완료** 클릭  
  4) SNS 로그인 계정이면 비밀번호 변경은 제한될 수 있음

- 질문 예시/답변 템플릿:
  - Q: 내 정보 수정은 어디서 해?
    - A: 좌측 메뉴 **유저 허브 > 내 정보 수정**에서 이름/생년월일 등을 수정한 뒤 **수정 완료**를 누르면 돼요.
  - Q: 비밀번호 변경이 안 돼요
    - A: SNS로 로그인한 계정은 화면에 안내된 것처럼 비밀번호 변경이 제한될 수 있어요.

- 키워드: 내 정보 수정, 유저 허브, 이름, 생년월일, 아이디, 이메일, 비밀번호 변경, SNS 계정, 수정 완료

## 레시피 생성하기 (AI로 생성하기)
- 메뉴 경로: 좌측 메뉴 > **레시피 생성하기** > **AI로 생성하기**
- 화면 설명:
  - 상단 제목: **AI 레시피 생성하기**
  - AI가 레시피를 만들어주는 영역(입력/대화 UI)이 배치됨

- 사용 방법(일반):
  1) 좌측 메뉴에서 **레시피 생성하기** 펼치기  
  2) **AI로 생성하기** 클릭  
  3) 화면 안내에 따라 원하는 조건을 입력/선택  
  4) 생성된 레시피를 확인하고 필요하면 수정/재생성

- 질문 예시/답변 템플릿:
  - Q: AI로 레시피 생성은 어디서 해?
    - A: 좌측 메뉴 **레시피 생성하기 > AI로 생성하기**로 들어가면 “AI 레시피 생성하기” 화면에서 만들 수 있어요.

- 키워드: AI 레시피 생성, AI로 생성하기, 레시피 생성하기, 생성, 재생성, 조건 입력

## 레시피 생성하기 (직접 등록하기)
- 메뉴 경로: 좌측 메뉴 > **레시피 생성하기** > **직접 등록하기**
- 화면 설명(프로젝트 흐름 기준):
  - 레시피를 사용자가 직접 등록하는 흐름(보통 단계형)
  - 포함 정보: **기본정보 / 재료 / 이미지 / 공개여부** 등

- 사용 방법(권장 흐름):
  1) **직접 등록하기** 클릭  
  2) 기본정보(레시피명/설명 등) 입력  
  3) 재료 입력  
  4) 이미지 업로드/등록  
  5) 공개 여부 선택 후 등록 완료

- 질문 예시/답변 템플릿:
  - Q: 레시피 직접 등록은 어디서 해?
    - A: 좌측 메뉴 **레시피 생성하기 > 직접 등록하기**에서 기본정보/재료/이미지/공개여부를 입력해서 등록할 수 있어요.

- 키워드: 직접 등록하기, 레시피 등록, 기본정보, 재료, 이미지, 공개여부, 저장, 등록 완료

## 레시피 상세 (레시피 정보 + 보고서 생성 설정)
- 진입 경로:
  - **레시피 허브(전체 레시피)** 에서 카드 클릭 → 레시피 상세
  - 또는 유저 레시피 목록에서 선택

- 화면 설명(스크린샷 기준 핵심):
  - 좌측: **레시피 정보**(이미지/소개/재료/조리 순서)
  - 상단/버튼: **공개**, **삭제** 등 관리 버튼이 있을 수 있음
  - 우측: **레시피 보고서**
    - “등록된 리포트가 없습니다” 같은 상태 문구가 뜰 수 있음
    - 하단에 **리포트 타겟 설정**(국가/페르소나/가격대/생성 옵션)
    - “리포트 옵션 선택(체크박스)” 영역:
      - 예: Global Market Map, 핵심 요약(필수), 시장 스냅샷(필수), 리스크&대응(필수),
            SWOT, KPI, 알레르기 성분 노트, 국가 수출 부적합 사례,
            컨셉 아이디어(필수), 제품 개발 추천안, 최종 요약(필수) 등
    - **보고서 공개 여부(비공개/공개)** 선택
    - **보고서 생성** 버튼

- 보고서 생성 방법(정확한 순서):
  1) 레시피 상세로 이동  
  2) 우측 **리포트 타겟 설정**에서
     - 국가 / 페르소나 / 가격대 / 생성 옵션을 선택  
  3) 아래 **리포트 옵션(섹션 체크박스)** 에서 포함할 항목 선택  
     - (필수 항목은 기본 포함되는 경우가 많음)  
  4) **공개 여부(비공개/공개)** 선택  
  5) **보고서 생성** 클릭  
  6) 생성이 끝나면 “레시피 보고서” 목록에 보고서가 추가됨(또는 리포트 목록에서 확인)

- 질문 예시/답변 템플릿:
  - Q: 보고서는 어디서 만들어?
    - A: **레시피 상세 화면** 오른쪽에 있는 “레시피 보고서 > 리포트 타겟 설정”에서 국가/페르소나/가격대/옵션을 선택하고, 아래 섹션 체크 후 **보고서 생성**을 누르면 돼요.
  - Q: 알레르기/수출부적합/인플루언서 같은 항목은 어디서 켜?
    - A: 레시피 상세 오른쪽의 **리포트 옵션(체크박스)** 영역에서 해당 섹션을 체크하면 보고서에 포함돼요.

- 키워드: 레시피 상세, 레시피 정보, 레시피 보고서, 보고서 생성, 리포트 타겟 설정, 국가, 페르소나, 가격대, 생성 옵션, 체크박스, 비공개, 공개, 삭제, 공개 버튼

## 최종 레시피 선정 (레시피 리포트 목록)
- 메뉴 경로: 좌측 메뉴 > **최종 레시피 선정** > **최종 레시피 선정**
- 화면 설명:
  - 상단 제목: **레시피 리포트 목록**
  - 검색창: “레시피 이름으로 리포트를 검색합니다”
  - 카드 형태로 리포트(요약/썸네일/레시피명)가 표시
  - 우측 상단 버튼: **보고서 선택하기**
  - 하단에 “최종 보고서 목록” 섹션이 있고, 없으면 “등록된 최종 보고서가 없습니다.” 표시

- 사용 방법:
  1) 좌측 메뉴 **최종 레시피 선정 > 최종 레시피 선정** 이동  
  2) 리포트 카드에서 원하는 리포트를 찾고(검색 가능)  
  3) **보고서 선택하기**로 최종 후보를 선택(프로젝트 로직에 따라 선택 화면/동작)  
  4) 선택 결과가 “최종 보고서 목록”에 쌓이거나 다음 분석 화면으로 이동

- 질문 예시/답변 템플릿:
  - Q: 최종 레시피 선정은 어디서 해?
    - A: 좌측 메뉴 **최종 레시피 선정 > 최종 레시피 선정**으로 들어가면 “레시피 리포트 목록”이 나오고, 우측 상단 **보고서 선택하기**로 최종 후보를 고를 수 있어요.

- 키워드: 최종 레시피 선정, 레시피 리포트 목록, 보고서 선택하기, 최종 보고서 목록, 리포트 검색, 카드

## 최종 평가 보고서 (AI 비교 분석 결과)
- 진입 경로(일반):
  - “최종 레시피 선정(레시피 리포트 목록)”에서 보고서를 선택/확정한 뒤
  - **AI 비교 분석 결과**(최종 평가 보고서) 화면으로 이동

- 화면 설명(스크린샷 기준):
  - 상단 제목: **AI 비교 분석 결과**
  - 좌측: “AI 심사위원 평가 점수” 레이더 차트(총점/맛/가격/건강)
  - 하단: 국가별 **총점/맛/가격/건강** 막대 그래프 섹션
  - 우측: 최종 추천 레시피, 선택 이유(핵심 근거), 비교 요약,
         KPI 분석 요약, SWOT 요약, 리스크 및 보완 제안 등 텍스트 리포트

- 질문 예시/답변 템플릿:
  - Q: AI 비교 분석 결과는 어디서 봐?
    - A: 좌측 메뉴 **최종 레시피 선정 > 최종 레시피 선정**에서 보고서를 선택/확정하면 “AI 비교 분석 결과” 화면에서 점수/그래프/요약을 확인할 수 있어요.
  - Q: 왜 이 레시피가 추천됐는지 보고 싶어
    - A: “AI 비교 분석 결과” 화면 오른쪽에 **선택 이유(핵심 근거)** 와 **비교 요약**이 정리돼 있어요.

- 키워드: AI 비교 분석 결과, 최종 평가 보고서, 레이더 차트, 총점, 맛, 가격, 건강, 국가별 점수, KPI, SWOT, 리스크, 보완 제안

## 보고서 상세 (섹션 구성) + PDF 다운로드
- 진입 경로(일반):
  - 레시피 상세의 “레시피 보고서” 목록에서 보고서 열기
  - 또는 최종/리포트 목록에서 보고서 클릭

- 화면 설명(스크린샷 기준 섹션 예시):
  - Global Market Map(지도)
  - 핵심 요약
  - 시장 스냅샷
  - 인플루언서 이미지/추천
  - 리스크 & 대응
  - SWOT
  - KPI 제안
  - 알레르기 성분 노트
  - 국가 수출 부적합 사례
  - 컨셉 아이디어
  - 제품 개발 추천안
  - 최종 요약
  - 하단에 **PDF 다운로드** 버튼

- PDF 다운로드 방법:
  1) 보고서 상세 화면으로 이동  
  2) 화면 하단(또는 버튼 영역)에서 **PDF 다운로드** 클릭  
  3) 다운로드된 PDF 파일을 확인

- 질문 예시/답변 템플릿:
  - Q: 보고서(PDF) 다운로드는 어디서 해?
    - A: 보고서 상세 화면에서 하단 **PDF 다운로드** 버튼을 누르면 돼요.
  - Q: 보고서에 어떤 내용이 들어가?
    - A: Global Market Map/핵심 요약/시장 스냅샷/인플루언서/리스크&대응/SWOT/KPI/알레르기/수출 부적합 사례/컨셉 아이디어/제품 개발 추천/최종 요약 등 섹션으로 구성돼요(선택한 옵션에 따라 달라질 수 있어요).

- 키워드: 보고서 상세, PDF 다운로드, 리포트 섹션, 글로벌 마켓 맵, 시장 스냅샷, 인플루언서, 리스크 대응, SWOT, KPI, 알레르기, 수출 부적합, 컨셉 아이디어, 최종 요약
